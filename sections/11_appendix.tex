\section{AI Tool Group Search Prompts}
\label{app:search_prompts}

The following N prompts were submitted verbatim to tool name to retrieve
candidate papers for Groups G1--G6 of the pre-validated corpus described in
Section~\ref{sec:methodology:corpora}. Tool name was used exclusively for
candidate \textit{retrieval}; all inclusion and exclusion decisions were made
by the author through manual assessment of each returned record.
